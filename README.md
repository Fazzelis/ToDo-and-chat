## Backend для сайта ToDo и чата
Данный проект был разработан в ходе дисциплины "Разработка интернет приложений".
Были реализованы возможности регистрации и входа в аккаунт. Также реализована работа с задачами и чатом.
## О проекте
В данном проекте было реализованно:
- Регистрация и вход в аккаунт с использованием jwt;
- Управление своими задачами (создание, изменение состояния, удаление задачи, удаление всех задач);
- Выход из аккаунта;
- Подключение к общему чату;
- Отправление сообщения в чат;
- Выход из чата.
## Стек технологий
- Python;
- FastAPI;
- sqlalchemy;
- alembic;
- jwt;
- REST API;
- pydantic;
- PostgreSQL;
- Docker;
- Websocket.
